连接 cjc npm私服

```javascript
安装nrm 工具： npm i nrm@1.0.0 -g

切换源： nrm add cjc https://packages.aliyun.com/68cba88215dfc6c8604f7adc/npm/repo-pncpo/

执行切换到新的源： nrm use cjc
```


下载私库组件示例：

```javascript
pnpm install @cjc/vue3-echarts
```
