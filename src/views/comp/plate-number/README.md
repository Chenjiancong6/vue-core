# 用于手机端录入车牌号码的组件

## 组件安装

```
pnpm i @cjc/vue3-plate-number
```

## 组件props

| 名称       | 类型   | 默认值                              | 说明                                                               |
| ---------- | ------ | ----------------------------------- | ------------------------------------------------------------------ |
| modelValue | String | ''                                  | 车牌号码，例如：`粤BHG3892`、`粤BE988F`，支持新能源车牌        |
| provinces  | Array  | [`['粤', '京',],['贵', '云']`]    | 输入面板车牌的省份数据，数据格式为数组包裹数组格式                 |
| numbers    | Array  | [`['1', '2', '3'`],`['Q', 'W']` | 输入面板车牌的数字和字母组合                                       |
| query      | 事件   |                                     | 函数接收一个返回参数，点击键盘的 `完成 按钮，返回车牌的绑定数据` |
